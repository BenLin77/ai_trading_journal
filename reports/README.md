# 📝 研究報告資料夾

此資料夾用於存放你的市場研究報告，供「投資組合 AI 顧問」分析使用。

## 📋 使用說明

1. **建立報告**：將你的 Markdown 研究報告放入此資料夾
2. **命名規範**：建議使用清晰的命名，如 `YYYYMMDD_標的_主題.md`
3. **格式自由**：可以使用任何 Markdown 格式，AI 會自動讀取

## ✍️ 報告範例

已提供 `EXAMPLE_Market_Analysis.md` 作為參考範本。

## 🤖 AI 如何使用這些報告

當你執行「投資組合 AI 顧問」時，AI 會：
1. 讀取你選擇的研究報告
2. 結合你的**實際持倉數據**
3. 整合**即時市場行情**
4. 提供具體可執行的避險與調整建議

## 💡 最佳實踐

- **定期更新**：每週至少更新一次主要持倉的分析
- **標注風險**：在報告中明確標注風險點和觸發條件
- **量化指標**：盡量提供具體的價格區間、比例建議
- **情境分析**：列出多種市場情境與對應策略

## 🔍 報告結構建議

```markdown
# 標題

## 市場概況
（總體經濟、市場情緒）

## 個股分析
### 標的 A
- 技術面分析
- 基本面分析
- 風險評估
- 操作建議

### 標的 B
...

## 投資組合建議
（整體配置、風險控制）

## 後市展望
（短中長期看法）
```

## ⚠️ 注意事項

- 報告僅供 AI 分析參考，不構成投資建議
- 最終決策需由你自行判斷並承擔風險
- 建議搭配其他技術分析工具綜合判斷
